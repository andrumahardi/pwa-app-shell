This app illustrates how to implement a PWA app that open sites hosted on subdomains or different domain names without showing the browser UI using "Shell Pattern"

This website has 3 sections:

  • The "Home" section resides at the following domain: https://andrumahardi.github.io/pwa-app-shell/
  • The "Original" link directs users to https://dev.to
  • The "Shell Pattern" enables the native experience of a different domain site (https://dev.to) within the Content Shell, seamlessly embedded within the overarching App Shell (https://andrumahardi.github.io/pwa-app-shell/)
